# engine

> [!IMPORTANT]
> **This project is my personal weapon as a solo freelance developer.**
>
> I built `engine` to amplify my own capacity: generate repetitive application code faster, keep delivery velocity high, and let me focus my energy on business logic and product decisions instead of boilerplate.
>
> It is first and foremost a personal productivity tool, shaped around my workflow, my DSL, and the way I build projects independently.

## Project notes

- Maven project; the authoritative project version is declared in `pom.xml`.
- `pom.xml` compiles the project with Java 25 and sets the jar manifest main class to `dev.cruding.engine.App`.
- Use `mvn test` to validate the project.
